# APCSA Labs

Java lab projects from my AP Computer Science A course (2023–2024), organized by unit. Each folder contains the source files for that lab.

## Unit 1 — Primitive Types & String Operations
| Lab | Topic |
|-----|-------|
| Unit1_Lab1_SphereCalculator | Computes diameter, circumference, surface area, and volume from a given radius |
| Unit1_Lab2_NameParser | Parses a full name string into first, middle, and last components |

## Unit 2 — Selection & Iteration
| Lab | Topic |
|-----|-------|
| Unit2_Lab1_LeapYear | Determines whether a given year is a leap year |
| Unit2_Lab2_Multiples | Prints multiples of a user-entered number using a while loop |

## Unit 3 — Object-Oriented Programming
| Lab | Topic |
|-----|-------|
| Unit3_Lab1_CherryPicker | Two-player cherry-picking game using a spinner class |
| Unit3_Lab2_SlotMachine | Slot machine simulation with a player and casino class |

## Unit 4 — Arrays
| Lab | Topic |
|-----|-------|
| Unit4_Lab1_SegregatedBinary | Segregates a random array of 0s and 1s (0s left, 1s right) |
| Unit4_Lab2_BankSystem | Bank simulation with deposit, withdrawal, and account management |

## Unit 5 — 2D Arrays
| Lab | Topic |
|-----|-------|
| Unit5_Lab1_FindModes | Finds the mode(s) of a randomly generated sequence using a sorted NumEntry array |
| Unit5_Lab2_MemoryGame | Console-based memory card matching game using a 2D array |

## Unit 6 — Inheritance
| Lab | Topic |
|-----|-------|
| Unit6_Lab1_SchoolAssociates | Class hierarchy for school associates (staff, teachers, students, professors) |
| Unit6_Lab2_GradeableInterface | Interface-based grading system with drivers permit and AP exam taker classes |

## Unit 7 — ArrayList
| Lab | Topic |
|-----|-------|
| Unit7_Lab1_AirlineTickets | Airline ticket manager using ArrayLists |
| Unit7_Lab2_Blackjack | Dealer vs. player Blackjack game with a full deck |

## Unit 8 — Recursion
| Lab | Topic |
|-----|-------|
| Unit8_Lab1_RecursiveMaximum | Finds the maximum value in an array recursively |
| Unit8_Lab2_BacteriaLab | Models bacterial growth recursively based on user input |

## Quarter Projects
| Project | Topic |
|---------|-------|
| Q1_Project_MadLibs | Console Mad Libs game with a Mongolian Empire theme |
| Q3_Project_InfectionSimulation | Graphical infection spread simulation using Java Swing |

## Running

Labs are plain Java (no build system). From any lab folder:

```
javac *.java
java <DriverClassName>
```
